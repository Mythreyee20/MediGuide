# 🏥 MediGuide

## AI-Powered Multilingual Hospital Navigation and Patient Helpdesk

MediGuide is an AI-powered healthcare assistance platform designed to help patients navigate hospitals more efficiently. The system provides department recommendations based on symptoms and offers guidance in both Tamil and English.

## 🚀 Problem Statement

Many patients face difficulties in:
- Identifying the correct hospital department
- Understanding hospital procedures
- Overcoming language barriers
- Getting quick guidance in crowded hospitals

## 💡 Solution

MediGuide acts as a virtual hospital assistant that:
- Analyzes user symptoms
- Recommends the appropriate department
- Provides multilingual support (Tamil & English)
- Answers patient queries through an AI chatbot
- Offers department-specific FAQs and recommended next steps

## ✨ Features

- Symptom-to-Department Recommendation
- AI Chat Assistant
- Tamil & English Language Support
- Department-Specific FAQ Section
- Recommended Next Steps Guidance
- User-Friendly Interface

- Enhances healthcare navigation experience
